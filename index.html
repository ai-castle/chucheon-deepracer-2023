대회 안내

본 대회는 춘천시에서 주최하는 AWS DeepRacer를 활용한 인공지능 강화학습 기반의 자율주행 자동차 AI 융합경진대회입니다.

  • 참가 대상
        - 인공지능 / 자율주행에 관심있는 전국 대학생 및 성인
        - 팀별 신청 (2~4명)
        - 파이썬 및 딥레이서 유경험자 우대
  • 참가 신청
        - 접수 기간 : 2023년 8월 23일 (수) ~ 9월 1일(금), 12:00 까지
        - 접수 방법 : 온라인 접수 (bit.ly/ai-chuncheon)
        - 모집 규모 : 30개팀 내외
  • 경기 방식
        - 예선 : 30개팀 / 온라인 대회
        - 본선 : 6개팀 / 오프라인 대회 (춘천시 커먼즈필드)
  • 지원
      - DeepRacer 계정 사용료 : 예선 (40시간), 본선 (30시간)
      - DeepRacer 교육 : 예선 및 본선 각 4시간
      - 딥레이서 차량 대여 (본선)
  • 시상 및 상품
    - 대상 (1팀) : 250만원 / 춘천시장상 / AWS 리인벤트 참관지원 예정
      * AWS 리인벤트 참관지원(2~3명)은 내부사정 및 현지상황에 따라 변경 또는 취소될 수 있음
    - 최우수상 (1팀) : 200만원 / 춘천시장상
    - 우수상 (2팀) : 100만원 / 강원정보문화산업진흥원장상
    - 장려상 (2팀) : 50만원 / 강원정보문화산업진흥원장상
  • 주최 및 주관
      - 주최 : 춘천시
      - 주관 : 강원정보문화산업진흥원, 에이아이캐슬
      - 후원 : AWS (Amazon Web Services)

예선 안내

예선은 딥레이서 온라인 시뮬레이션에서 진행됩니다. 예선 기간 내에 제출한 모델 중 가장 우수한 기록을 기준으로 순위를 평가합니다. 예선에서 우수한 성적을 거둔 상위 6개 팀은 본선 참가 자격을 얻습니다. 예선에 대한 자세한 내용은 예선 OT에서 안내됩니다.

- 예선 기간 : 09/02 (토) - 09/08 (금), 12:00 모델 제출 마감
- 지원 : AWS 계정 및 40시간의 모델 훈련 시간
- 예선전 규칙 (pw 별도 안내)

본선 안내

본선은 오프라인 실제 트랙에서 진행되며, 예선에서 우수한 성적을 거둔 상위 6개 팀이 참가 자격을 가집니다. 각 팀별로 20분씩 2회, 총 40분의 시간이 주어집니다. 주어진 시간 내에 한 바퀴를 가장 빠르게 완주한 기록을 팀의 최종 기록으로 합니다. 본선에 대한 자세한 내용은 본선 OT에서 안내됩니다.

- 본선 날짜 : 09/17 (일), 10:00 - 17:00
- 본선 장소 : 춘천 커먼즈필드 안녕하우스 (강원 춘천시 공지로 255)
- 지원 : AWS 계정 및 30시간의 모델 훈련 시간 / 숙박 (2박 3일) 및 식사 / 딥레이서 차량 대여
- 본선 시간표
- 본선 규칙 (pw 별도 안내)

테스트 주행 (본선)

본선에 참가하는 6개 팀은 본선 대회 전 이틀간 진행되는 테스트 주행에 참가하여 준비를 해야합니다. 한 팀당 50분씩 이틀간 총 100분의 테스트 주행 시간이 부여됩니다. 테스트 주행에 대한 자세한 내용은 본선 OT에서 안내됩니다.

- 장소 : 본선 장소와 동일 (춘천 커먼즈필드 안녕하우스)
- 지원 : 숙박 (2박 3일) 및 식사 제공 / 딥레이서 차량 대여
- 테스트 주행1 (9/15, 금) : 팀원 중 최소 한명 참석
- 테스트 주행1 (9/16, 토) : 팀원 전원 필수 참석
- 테스트 주행 시간표

숙소 안내 (본선)

본선에 참가하는 팀에게는 테스트 주행 (9/15,16) 및 본선 (9/17) 기간 동안 춘천에 머무를 수 있는 숙소를 2박 3일간 지원합니다. 테스트 주행에 대한 자세한 내용은 본선 OT에서 안내됩니다.

- 숙소 장소 : 소락재 (강원 춘천시 공지로333번길 8)
- 숙박 기간 : 9/15 - 17 (2박 3일)
- 팀별 1 객실 제공 (4인실)
- 네이버 지도
- 혼성으로 구성된 팀의 경우 별도의 도미토리 객실을 추가로 제공할 예정입니다.

식사 안내 (본선)

본선에 참가하는 팀에게는 테스트 주행 (9/15,16) 및 본선 (9/17) 기간 동안 식사 및 음료 (카페)를 지원합니다.

- 테스트 주행 (9/15, 16) 저녁 식사 : 숙소(소락재)에서 제공
- 본선 대회 당일 (9/17) 점심 식사 : 태화루 (예정)
- 본선 대회 당일 (9/17) 음료 제공 : 커먼즈필드 내 카페 (WLCM)
